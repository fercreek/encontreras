Conoce la arquitectura detrás de Encontreras y cómo usa Playwright y LLMs para revolucionar el Outbound B2B.

🚨 El Problema del Scraping Tradicional

La mayoría de los vendedores B2B compran bases de datos frías o usan extractores genéricos. El resultado es siempre el mismo: una tabla de Excel con miles de nombres y teléfonos que nadie contesta.

Encontreras piensa diferente. No queremos un millón de teléfonos inútiles. Queremos 50 prospectos de alta calidad, con contexto real de su negocio, sabiendo exactamente dónde les duele (¿su web está caída? ¿nadie los sigue en redes?). Ese contexto es la diferencia entre un vendedor pesado y un consultor valioso.

🛠️ ¿Por qué construirlo así?

¿Por qué Playwright y no Requests/BeautifulSoup solos?
Google Maps renderiza toda su data dinámicamente con JavaScript. Una simple petición HTTP devuelve una página en blanco. Playwright levanta un Chromium real que ejecuta JS y engaña los métodos anti-scraping simulando scrolls y clicks.

¿Por qué SQLite?
Almacenar en JSON está bien para 10 registros, pero cuando pasas de 500, necesitas un motor de BD real para deduplicar eficientemente con UPSERT (ON CONFLICT DO UPDATE) y para servir la API del Dashboard en tiempo real a alta velocidad.

¿Por qué procesar en background (Worker / Huey)?
Ir a Google Maps, entrar a 10 páginas web, navegar por Instagram y consultar a una IA toma mucho **tiempo IO**. Si lo hiciéramos bloqueando el servidor HTTP, el Dashboard tardaría 3 minutos en cargar una página web. Huey procesa todo asíncronamente en segundo plano, manteniendo el UI fluido.

💡 ¿Qué puedes vender con estos datos?

  • Agencia SEO: Filtra los negocios con Web con Problemas. Mándales el DM "Directo" diciendo que su página está caída.
  • Agencia de Redes Sociales: Busca negocios con 5 estrellas en Maps, pero con 0 seguidores en Instagram. Escríbeles el DM "Halago" destacando que su excelente servicio en persona no se refleja en digital.
  • Venta de Software (SaaS): Analiza los horarios de miles de clínicas y véndeles un sistema de reservas si ves que no tienen botón de agendar en su web.
